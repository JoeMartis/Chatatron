The Opportunity

Quality at the scale UAI demands

As the UAI program grows, so does the volume of modules and the complexity of the quality standards behind them. Fixatron automates the tedious parts so the focus stays on what matters most — the learning experience.

  • Growing volume of modules — more content means more to review; automation lets the team keep pace without cutting corners
  • Hundreds of settings per module — XML metadata, naming conventions, accessibility attributes — automation catches every detail instantly
  • Consistency across the catalog — a single automated standard ensures every module is measured the same way
  • Faster feedback loops — content authors can check quality instantly, accelerating the path from draft to published

The Story

Built through human + AI partnership

Fixatron wasn't built by AI alone. It was a true collaboration — iterating together through real UAI Module exports, discovering patterns, refining rules, and pushing toward higher quality.

1

Studied real UAI Module exports

We started by examining actual module exports together — understanding the structure, the style guide expectations, and where issues tend to hide.

2

Wrote rules, tested against real content, iterated

Each rule was crafted, tested against multiple module exports, and refined. When a rule was too strict or too lenient, we adjusted together until it matched real-world expectations.

3

Assigned specialized AI agents to different concerns

Security, bug detection, UI/UX, user flow, accessibility — each area got focused attention from dedicated AI agents, working in parallel and then integrated.

4

Continuous refinement through real-world testing

We kept running new modules through the tool, discovering edge cases, and expanding coverage — growing from a handful of checks to 95 comprehensive rules.

AI at Work

Multiple AI agents, one mission

We didn't just use AI — we orchestrated multiple specialized agents, each focused on a different dimension of quality. This is what thoughtful AI collaboration looks like.

🛡️

Security Agent

Ensured safe file handling, prompt injection defense, and no data leakage — because UAI content must be protected.

🐛

Bug Detection Agent

Hunted for edge cases in XML parsing, rule logic, and fix operations — catching issues before they reach users.

🎨

UI/UX Agent

Designed the interface so anyone can understand findings at a glance and take action with one click.

🔄

User Flow Agent

Optimized the end-to-end workflow: upload, review findings, apply fixes, export — simple and intuitive.

🧠

Content Quality Agent

Uses Claude to evaluate whether explanations are substantive, terminology is consistent, and learning outcomes are met.

🧪

Testing Agent

Built a comprehensive test suite — 254 tests ensuring every rule, fix, and integration works correctly.
